# 🌦️ Weather App

A simple and responsive weather application that allows users to check the current weather conditions for any location.

## 📌 Features

* Search for weather information by location
* Fetch real-time weather data from an API
* Display temperature, weather condition, and location
* Dynamic UI that changes based on weather conditions
* Clean and modern design using **Tailwind CSS**

## 🛠️ Technologies Used

* HTML
* CSS
* Tailwind CSS
* JavaScript
* Weather API

## 🚀 How It Works

1. The user enters a location in the input field.
2. The application sends a request to a weather API.
3. The API returns weather data in JSON format.
4. JavaScript processes the data and displays it on the page.
5. The UI updates dynamically based on the weather condition.
